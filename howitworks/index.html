How it works

Collecting the best web apps on the web and link them.

We and our community search for the best web apps for small tasks, like generators, converters, or other tools. Then we choose the best one of every kind and award this page by including them in our network. This means we link to them, and other award-winning web apps too. With this, we want to achieve that users can find a reliable web app if they look for it quickly.

The link

apuem.com/[Language code]/[Web app name]

e.g. apuem.com/en/loremipsum
